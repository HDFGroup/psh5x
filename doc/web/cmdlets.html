PSH5X Cmdlets

The purpose of the table below is to provide an overview of the cmdlets available in PSH5X. Detailed help is provided with each cmdlet via the PowerShell help system.

Cmdlet Description
Copy-...
Copy-H5Attribute Copies HDF5 attributes of objects on the same or different HDF5 drives. This is an alias for the core provider's Copy-ItemProperty cmdlet.
Copy-H5Object Copies HDF5 objects (dataset, group, linked datatype) on the same or between different HDF5 drives.
Copy-H5SymbolicLink Copies HDF5 symbolic links (soft link and external link) on the same or between different HDF5 drives.
Format-H5Dataset Changes the extent (dimension sizes) of extendible HDF5 datasets.
Get-...
Get-H5Attribute Gets a representation of an HDF5 attribute. This is an alias for the core provider's Get-ItemProperty cmdlet.
Get-H5DatasetValue Gets a, generally multidimensional, .NET array representing the value of an HDF5 dataset or a hyperslab selection.
Get-H5Drive Get a representation of an HDF5 drive (H5Drive).
Get-H5Member Gets a list representing the links in an HDF5 group (group members). This is an alias for the core provider's Get-ChildItem cmdlet.
Get-H5Item Gets a representation of an HDF5 item (HDF5 object or symbolic link). This is an alias for the core provider's Get-Item cmdlet.
Move-...
Move-H5Attribute Moves an HDF5 attribute between HDF5 objects on the same or different HDF5 drives. This is an alias for the core provider's Move-ItemProperty cmdlet.
Move-H5Item Moves an HDF5 item to a new location on the same HDF5 drive. (Nothing is really moved. Link acrobatics only.) This is an alias for the core provider's Move-Item cmdlet.
New-...
New-H5Attribute Creates a new HDF5 attribute on an HDF5 object.
New-H5Dataset Creates a new HDF5 dataset with a simple dataspace.
New-H5Drive Create a new HDF5 drive backed by an HDF5 file.
New-H5File Create a new HDF5 file.
New-H5Group Creates a new HDF5 group.
New-H5Hardlink Creates a new hard link to an HDF5 object in an HDF5 group.
New-H5Image Creates a new HDF5 image from an image file. (See HDF5 Image and Palette Specification and gif2h5.)
New-H5LinkedDatatype Creates a new HDF5 linked (named) datatype object.
New-H5NullDataset Creates a new HDF5 dataset with a null dataspace (empty set).
New-H5ScalarDataset Creates a new HDF5 dataset with a scalar dataspace (singleton).
New-H5Softlink Creates a soft link in an HDF5 group.
Remove-...
Remove-H5Attribute Removes an HDF5 attribute from an HDF5 object. This is an alias for the core provider's Remove-ItemProperty cmdlet.
Remove-H5Item Removes and HDF5 item (HDF5 object or symbolic link). This cmdlet sounds grander than what it really does: All it does is remove links (hard or symbolic). This is an alias for the core provider's Remove-Item cmdlet.
Rename-...
Rename-H5Attribute Renames HDF5 attribute of an HDF5 object. This is an alias for the core provider's Rename-ItemProperty cmdlet.
Rename-H5Item Renames HDF5 symbloc links or a hard link to an HDF5 object. This is an alias for the core provider's Rename-Item cmdlet.
Set-...
Set-H5Attribute Sets the value of an HDF5 attribute. This is an alias for the core provider's Set-ItemProperty cmdlet.
Set-H5DatasetValue Sets the value of an HDF5 dataset or a hyperslab selection.
Test-...
Test-H5Attribute Tests if an HDF5 object has an HDF5 attribute of a given name.
Test-H5Path Tests if an HDF5 path name is in use or can be resolved to an HDF5 object. This is an alias for the core provider's Test-Path cmdlet.
