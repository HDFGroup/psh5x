PSH5X

PSH5X is a Windows PowerShell module for HDF5.

Prerequisites

Operating system:
  • Windows XP, Vista, 7, 8
  • Windows Server 2003, 2008, 2008 R2, 8
PowerShell:
  • Version 2.0
  • Version 3.0

Version 1.0 is not supported.

.NET Framework
  • Version 4
  • Version 4.5

Earlier versions are currently not supported. PSH5X may work with .NET 3.5, but this arrangement has not been tested.
