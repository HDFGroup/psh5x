TOPIC
    about_H5Bitfield

SHORT DESCRIPTION

    An HDF5 bitfield type defines instances that are fixed-length
    sequences of bits packed into one or more bytes.

SYNTAX

    <bit_padding> ::= Zero | One | Background

    <byte_order> ::= BE | LE

    @{
        BitOffset    = <non-negative integer> ;
        ByteOrder    = <byte_order> ;
        Class        = Bitfield ;
        LSBitPadding = <bit_padding> ;
        MSBitPadding = <bit_padding> ;
        Precision    = <positive integer> ;
        Size         = <positive integer> ;
    }

NOTES

    Predefined HDF5 bitfield types are:

        H5T_STD_B8BE
        H5T_STD_B8LE
        H5T_STD_B16BE 
        H5T_STD_B16LE
        H5T_STD_B32BE
        H5T_STD_B32LE
        H5T_STD_B64BE 
        H5T_STD_B64LE
